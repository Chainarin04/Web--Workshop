Workshop: HTML Heading

ฝึกใช้แท็ก <h1> – <h6> และการปรับขนาดหัวข้อด้วย CSS

01

ความหมายของ Headings (H1–H6)

HTML มีหัวข้อทั้งหมด 6 ระดับ

  • H1 – หัวข้อใหญ่สุด ใช้สำหรับชื่อหน้าเว็บหรือหัวข้อหลัก
  • H2 – หัวข้อรอง แบ่ง Section ของเนื้อหา
  • H3 – หัวข้อย่อยภายใน H2
  • H4 – ย่อยลงไปอีกระดับ
  • H5 – ใช้น้อย แต่เน้นลำดับข้อมูล
  • H6 – เล็กสุด ใช้กับหัวข้อระดับลึก
H1 ควรมีเพียง 1 อันในแต่ละหน้า เพื่อช่วย SEO และโครงสร้างเอกสาร
02

ตัวอย่างการใช้ H1–H6

<h1>Welcome to My Website</h1>
<h2>HTML Basics</h2>
<h3>What is HTML?</h3>
<h4>HTML Syntax</h4>
<h5>Example Code</h5>
<h6>Note: Case Sensitive</h6>

เมื่อแสดงผลจะได้ขนาดลดหลั่นแบบนี้:

H1: Welcome to My Website

H2: HTML Basics

H3: What is HTML?

H4: HTML Syntax

H5: Example Code
H6: Note about heading size
💡 Best Practices (ข้อแนะนำเพิ่มเติม)

Tip: ควรใช้แท็ก <h1> เพียงแค่ 1 ครั้งต่อ 1 หน้าเพจเท่านั้น — เพราะมันเปรียบเสมือน Main Topic หรือ Title หลักของหน้านั้นๆ

Note: ใช้ HTML headings สำหรับการจัดโครงสร้าง "หัวข้อ" เท่านั้น! Don't use headings to make text BIG or bold (อย่าใช้ Heading เพียงเพราะอยากให้ตัวหนังสือใหญ่หรือหนาขึ้นเด็ดขาด)
